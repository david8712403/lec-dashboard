# LEC Dashboard Web

Next.js App Router frontend for the LEC Dashboard.

## Development

From the repo root:

```bash
npm --workspace web run dev
```

Or use `npm run dev` at the repo root to start both web + server.

## Environment

Create `web/.env.local` if you need a custom backend URL:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:3001
```
