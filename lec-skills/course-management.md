# Course Management Skill 使用指南

## 📦 技能說明

這是一個完整的課程管理系統技能，專為教育機構設計，用於管理：
- 學員基本資料
- 課程安排（每週固定時段）
- 上課記錄與請假管理
- 檢測記錄（視聽動發展追蹤）

## 🚀 安裝方式

1. 在 Claude.ai 中，點擊右上角的設定圖示
2. 選擇「Skills」
3. 點擊「Upload Skill」
4. 上傳 `course-management.skill` 檔案

## ✨ 功能特色

### 1️⃣ 學員管理
- **類型分類**：a超前、b一般、c特殊
- **狀態追蹤**：檢測中、進行中、成案、離室
- **年齡計算**：自動計算學員當前年齡

### 2️⃣ 課程安排
- **固定時段**：支援每週固定課表
- **標準時段**：
  - 09:00-10:40、10:40-12:20
  - 13:00-15:00、15:00-17:00
  - 17:00-19:00、19:00-21:00
- **靈活查詢**：可查詢學員課表或週課表

### 3️⃣ 上課記錄
- **三大領域記錄**：視覺、聽覺、運動課程內容
- **出席狀態**：出席、請假、缺席
- **日期彈性**：支援"下週二"、"2/7"等多種格式

### 4️⃣ 檢測記錄
- **發展年齡追蹤**：記錄視聽動各項發展年齡（如 2-6 表示 2歲6個月）
- **課程比例規劃**：視覺、聽覺、運動、學科比例（總和100%）
- **進展比較**：自動比較初測與複測結果

## 📝 使用範例

安裝技能後，您可以這樣使用：

### 新增學員
```
建立一個個案A，生日是2020/04/03
```

### 安排課程
```
個案A固定週三13:00來上課
```

### 請假記錄
```
個案A下週二請假
個案A 2/7請假
```

### 課程備註
```
個案A下週要多加強視覺練習
```

### 檢測記錄
```
個案B 上週第一次檢測，視聽動個別是 2-4、3-1、4-1，課程比例分成視聽動+學科 40/30/0/30
```

### 複測記錄
```
個案B 2024/8/2的複測結果，視聽動個別是 3-2、3-8、4-6，課程比例分成視聽動+學科 30/30/10/30
```

### 查詢功能
```
查詢個案A的完整資料
顯示週三的課表
個案B的檢測進展如何？
```

## 🗄️ 資料庫結構

技能使用 SQLite 資料庫儲存資料，包含以下表格：

1. **students** - 學員基本資料
2. **schedules** - 固定課程安排
3. **attendance_records** - 上課記錄
4. **assessment_records** - 檢測記錄
5. **leave_records** - 請假記錄
6. **class_notes** - 課程備註

## 📊 資料持久性

- 技能會自動初始化並維護 SQLite 資料庫
- 所有資料都會永久保存在 `course_management.db` 檔案中
- 支援匯出資料為 Excel 或 CSV 格式（可直接要求 Claude 幫您匯出）
